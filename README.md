# A Town Limo - Premium Luxury Transportation

A modern, responsive website for A Town Limo, Atlanta's premier luxury transportation service.

## Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Interactive Booking Widget**: Real-time price estimation and availability
- **Premium Fleet Showcase**: 3D parallax gallery with vehicle details
- **Social Proof Elements**: Live booking notifications and trust badges
- **Professional UI/UX**: Modern design with smooth animations
- **SEO Optimized**: Meta tags and structured content

## Fleet

- **Executive Sedan**: Elegant comfort for business transfers (3 passengers)
- **Cadillac Escalade ESV**: Pinnacle of luxury SUV travel (6 passengers)
- **Mercedes Sprinter**: Refined comfort for group travel (14 passengers)

## Services

- Airport Transfers with flight tracking
- Corporate Travel for executives
- Special Events (weddings, proms, celebrations)
- Hourly Charters for flexible bookings
